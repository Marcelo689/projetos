Para usar a calculadora você deve possuir uma ide para compilar o seu projeto jframe do java no meu caso utilizei o Apache netbeans .
Depois deverá ter instalado o jdk
então basta abrir a ide e clicar em open project , então você selecionara a pasta Calculadora que retirou do github.
Ira abrir SourcePackages/com.mycompany.mavenproject2/Formulario.java;
E pressione Shift+F6
Pronto ! sua calculadora está aberta.
